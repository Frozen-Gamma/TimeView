TimeView
---------

A php project that allows you to see player time's when using a MySql database with TimeTrail.

Install
-------
-Place the TimeView folder on your webserver.
-Edit the config.php file for your MySql information, the constant time infomation and the Time's you are using.
-Thats it! Just go to the location of Timeview in your browser and voila! It should be there.

If you have any queries with TimeTrail or Timeview please let us know here: https://github.com/Frozen-Gamma/TimeTrail/issues
